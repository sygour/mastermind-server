Mastermind

You all know this game: find the combination the master has chosen.

The specificity, here, is that a new game is start regularly, if you found the combination you own points, otherwise you loose some.

API Description

Registering as a player

POST /mastermind/players?name=<YOUR_TEAM>

[{
  "id": your_unique_id,
  "name": YOUR_TEAM
}]

NB: keep this id, as you'll need it later on.

Getting current game settings

GET /mastermind

{
  "id": current_game_id,
  "size": number_of_pawns_in_game
  "colors": [
    available_color_1,
    ...,
    available_color_n
  ]
}

Proposing a solution

POST /mastermind/{current_game_id}/solutions
player: your_unique_id

[
  pawn_color_1,
  ...,
  pawn_color_m,
]

In case of error, an empty body is return and header "error" contains the error message. 

[
  {
    "result": [
      "MISS_PLACED",
      "INCORRECT",
      ...
      "CORRECT",
    ]
  }
]
